API Integration Instructions

To use the AUDAA Text-to-Speech service, follow these steps:

  1. Install the Zyphra client: 
    pip install zyphra
  2. Initialize the client: 
    from zyphra import ZyphraClient
client = ZyphraClient(api_key="your-api-key")
  3. Generate speech: 
    audio_data = client.audio.speech.create(
    text="Hello, world!",
    language_iso_code="en-us",
    mime_type="audio/mp3",
    speaking_rate=15
)

Features

  • Multiple language support

  • Multiple audio formats
  • Real-time preview
