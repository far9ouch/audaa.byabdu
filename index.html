AUDAA

Transform Text to Natural Speech

Professional text-to-speech conversion in multiple languages

Features

  • Multiple language support
  • Adjustable speaking rate
  • Multiple audio formats (MP3, WAV, WebM, OGG)
  • Real-time preview functionality

Installation

To use this service locally:

  1. Clone the repository:
    git clone https://github.com/far9ouch/audaa.byabdu.git
  2. Install requirements:
    pip install -r requirements.txt
  3. Run the Flask app:
    python app.py
  4. Visit http://localhost:5000 in your browser
